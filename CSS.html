CSS Kommando Huskeliste

¨

Velgelementer (Selectors)

*
Universalvelger – velger alle elementer

Detaljert Forklaring

  • Velger alle elementer på siden.

Eksempel

* {
    margin: 0;
    padding: 0;
}
element
Velger alle forekomster av et spesifikt HTML-element (f.eks. p eller h1)

Detaljert Forklaring

  • Når du angir et elementnavn, anvendes stilen på alle slike elementer.

Eksempel

p {
    font-size: 16px;
}
.klasse
Velger alle elementer med en bestemt klasse

Detaljert Forklaring

  • Brukes for å gjenbruke stiler på flere elementer.

Eksempel

.button {
    background-color: blue;
    color: white;
}
#id
Velger et unikt element med et spesifikt ID

Detaljert Forklaring

  • ID skal være unike for hvert element.

Eksempel

#header {
    background-color: #f0f0f0;
}

Grunnleggende Egenskaper

color
Angir tekstfarge

Detaljert Forklaring

  • Farge kan angis med navn, hex-verdier, RGB eller RGBA.

Eksempel

h1 {
    color: #9b4da3;
}
background-color
Angir bakgrunnsfarge

Detaljert Forklaring

  • Sørger for at elementet får ønsket bakgrunn.

Eksempel

body {
    background-color: #f8f0f9;
}
font-size
Angir tekststørrelse

Detaljert Forklaring

  • Størrelsen kan spesifiseres i px, em, rem, % osv.

Eksempel

p {
    font-size: 16px;
}
margin
Angir ytre avstand

Detaljert Forklaring

  • Kan settes med én, to, tre eller fire verdier.

Eksempel

.section {
    margin: 20px;
}
padding
Angir indre avstand

Detaljert Forklaring

  • Lager mellomrom mellom innhold og kantlinje.

Eksempel

.button {
    padding: 10px 15px;
}

Layout og Display

display
Angir elementets visningsmodus, f.eks. block, inline eller flex

Detaljert Forklaring

  • Kontrollerer hvordan elementer og deres barn plasseres i layouten.

Eksempel

nav {
    display: flex;
    justify-content: space-around;
}
position
Setter posisjoneringen (static, relative, absolute, fixed, sticky)

Detaljert Forklaring

  • Brukes til å plassere elementer nøyaktig der du ønsker.

Eksempel

.tooltip {
    position: absolute;
    top: 50px;
    left: 100px;
}
flex-direction
Angir retningen for elementer i en fleksbeholder (f.eks. row eller column)

Detaljert Forklaring

  • Endrer rekkefølgen på fleks-elementer.

Eksempel

.container {
    display: flex;
    flex-direction: column;
}

Media Queries

@media (max-width: 600px)
Tilpasser design for mindre skjermer

Detaljert Forklaring

  • Brukes for å lage responsive design.

Eksempel

@media (max-width: 600px) {
    body {
        background-color: lightgray;
    }
}

CSS Animasjoner

@keyframes
Definerer animasjonssekvenser

Detaljert Forklaring

  • Setter opp rammer for en animasjon, som kan brukes med animation-egenskapen.

Eksempel

@keyframes fadeIn {
    from { opacity: 0; }
    to   { opacity: 1; }
}

.element {
    animation: fadeIn 2s ease-in;
}

Pseudo-classes & Pseudo-elements

:hover
Styling som aktiveres ved hover-effekt

Detaljert Forklaring

  • Pseudo-klassen :hover endrer stil når brukeren peker med musen over et element.

Eksempel

a:hover {
    color: #E6C0E9;
}
::before
Legger til innhold før et element

Detaljert Forklaring

  • Brukes til å sette inn dekorative elementer uten å endre HTML-strukturen.

Eksempel

.button::before {
    content: "★";
    margin-right: 5px;
}

CSS Grid & Flexbox

display: grid;
Oppretter et grid-layout

Detaljert Forklaring

  • CSS Grid gir et fleksibelt og kraftig layoutsystem for komplekse design.

Eksempel

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 20px;
}
display: flex;
Oppretter et fleksibelt layout

Detaljert Forklaring

  • Med Flexbox kan du enkelt lage horisontale eller vertikale oppsett.

Eksempel

nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

CSS Variabler

:root { --primary-color: #9b4da3; }
Definerer globale CSS-variabler

Detaljert Forklaring

  • Variabler lar deg lagre verdier for farger, størrelser osv. som kan gjenbrukes over hele dokumentet.

Eksempel

:root {
    --primary-color: #9b4da3;
    --secondary-color: #E6C0E9;
}

h1 {
    color: var(--primary-color);
}

Transitions & Transforms

transition
Legger til jevne overganger

Detaljert Forklaring

  • Overgangseffekten angir hvordan en egenskap endres over tid.

Eksempel

.button {
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: var(--secondary-color);
}
transform
Endrer form og posisjon på et element

Detaljert Forklaring

  • Transform gir mulighet til funksjoner som rotate, scale, translate, med mer.

Eksempel

.card:hover {
    transform: scale(1.05) rotate(1deg);
}

Avanserte Selektorer

[attributtnavn]
Velger elementer med et bestemt attributt

Detaljert Forklaring

  • Velger alle elementer som har den spesifiserte attributten, uavhengig av verdien.

Eksempel

input[required] {
    border: 2px solid red;
}
[attributtnavn="verdi"]
Velger elementer med en spesifikk verdi for et attributt

Detaljert Forklaring

  • Brukes for å velge elementer hvor attributtet har en nøyaktig verdi.

Eksempel

a[href="https://www.example.com"] {
    color: green;
}
element + element
Velger det første elementet umiddelbart etter et spesifikt element

Detaljert Forklaring

  • Plukker det første søskenelementet som kommer like etter et gitt element.

Eksempel

h1 + p {
    margin-top: 10px;
}
element ~ element
Velger alle søskenelementer etter et spesifikt element

Detaljert Forklaring

  • Velger alle elementer som er søsken og kommer etter et gitt element, uavhengig av avstand.

Eksempel

h2 ~ p {
    color: gray;
}
:nth-child(n)
Velger element basert på deres posisjon blant søsken

Detaljert Forklaring

  • Brukes for å velge et element basert på dets rekkefølge, for eksempel å velge den andre raden i en liste.

Eksempel

li:nth-child(2) {
    background-color: lightblue;
}

CSS Borders & Shadows

border
Angir rammer rundt et element

Detaljert Forklaring

  • Setter bredde, stil og farge for rammen rundt et element.

Eksempel

.box {
    border: 2px solid #333;
}
border-radius
Gir elementet avrundede hjørner

Detaljert Forklaring

  • Brukes for å lage runde eller avrundede hjørner på et element.

Eksempel

.box {
    border-radius: 8px;
}
box-shadow
Legger til en skygge bak et element

Detaljert Forklaring

  • Gir en dybdeeffekt ved å legge til skygge rundt et element.

Eksempel

.card {
    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
}

CSS Filters

filter
Legger til visuelle effekter som blur, kontrast og lysstyrke

Detaljert Forklaring

  • Effekten anvendes for å manipulere utseendet på et element, for eksempel å gjøre et bilde grått eller uskarpt.

Eksempel

img {
    filter: grayscale(50%) blur(2px);
}

CSS Tekst & Typografi

font-family
Definerer skrifttype for tekst

Detaljert Forklaring

  • Setter skrifttype og fallback-liste for et element.

Eksempel

body {
    font-family: 'Helvetica Neue', Arial, sans-serif;
}
line-height
Angir linjeavstand for tekst

Detaljert Forklaring

  • Øker lesbarheten ved å definere vertikalt mellomrom mellom tekstlinjer.

Eksempel

p {
    line-height: 1.6;
}
text-shadow
Legger til skygge på teksten

Detaljert Forklaring

  • Skaper en 3D-effekt eller gir bedre kontrast ved å legge til en skygge bak teksten.

Eksempel

h1 {
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}

CSS Bakgrunn

background-image
Setter et bakgrunnsbilde for et element

Detaljert Forklaring

  • Brukes for å definere et bilde som bakgrunn på et element.

Eksempel

.header {
    background-image: url('/images/header-bg.jpg');
}
background-size
Angir størrelsen på bakgrunnsbildet

Detaljert Forklaring

  • Kan brukes for å dekke hele elementet (cover) eller tilpasse bildet til containeren (contain).

Eksempel

.header {
    background-size: cover;
}
background-position
Kontrollerer posisjoneringen av bakgrunnsbildet

Detaljert Forklaring

  • Bestemmer hvor bakgrunnsbildet skal plasseres i forhold til elementet.

Eksempel

.header {
    background-position: center center;
}
background-repeat
Angir om bakgrunnsbildet skal repeteres

Detaljert Forklaring

  • Definerer om bakgrunnsbildet skal gjentas (repeat), ikke gjentas (no-repeat) eller gjentas bare horisontalt/vertikalt.

Eksempel

.header {
    background-repeat: no-repeat;
}

CSS Box Model

box-sizing
Definerer hvordan bredde og høyde beregnes for et element

Detaljert Forklaring

  • Med border-box inkluderer elementet padding og border i den totale bredden og høyden.

Eksempel

* {
    box-sizing: border-box;
}
overflow
Kontrollerer visningen av overflødig innhold

Detaljert Forklaring

  • Bruk verdiene hidden, scroll eller auto for å styre hvordan innhold utenfor et elements dimensjoner vises.

Eksempel

.container {
    overflow: auto;
}
z-index
Angir stablingsrekkefølgen til posisjonerte elementer

Detaljert Forklaring

  • Elementer med høyere z-index vises foran elementer med lavere verdi.

Eksempel

.modal {
    position: fixed;
    z-index: 1000;
}

CSS Avanserte Effekter

clip-path
Definerer en klippeflate for et element

Detaljert Forklaring

  • Begrenser synligheten til en bestemt form på elementet.

Eksempel

.image {
    clip-path: circle(50%);
}
mix-blend-mode
Angir hvordan et element skal blandes med bakgrunnen

Detaljert Forklaring

  • Skaper visuelle effekter ved å endre på hvordan farger kombineres med underliggende elementer.

Eksempel

.overlay {
    mix-blend-mode: multiply;
}
object-fit
Bestemmer hvordan medieinnhold (f.eks. bilder eller video) skal tilpasse seg containeren

Detaljert Forklaring

  • Med verdiene cover eller contain kan du styre om mediet skal kuttes eller tilpasses helt innenfor containeren.

Eksempel

img {
    width: 300px;
    height: 200px;
    object-fit: cover;
}
