# KodeList

KodeList er en ressurs som viser hvordan lister kan defineres og brukes i forskjellige programmeringsspråk. Dette prosjektet har som mål å lage en norsk huskeliste over kode som er lett å forstå og bruke.

## Innhold

 Liste over: 
 HTML
 CSS
 Javascript
 Python
 SQL
 GIT
 GO
 CSharp
 NodeJS
 React
 Rust
 Typescript
 Vue

### Eksempler på Git-kommandoer i JSON-format

Git-kommandoer blir presentert i et JSON-format som gjør det enkelt å strukturere og organisere informasjon. Nedenfor er en enkel oversikt over hvordan en JSON-fil kan se ut, med en kategori for grunnleggende Git-kommandoer og kommandoer som er relatert til fork.

```json
{
  "categories": [
    {
      "name": "Grunnleggende",
      "commands": [
        {
          "syntax": "git init",
          "description": "Oppretter et nytt git-repositorium",
          "details": "<strong>git init:</strong> Initialiserer et nytt, tomt git-repo i gjeldende mappe."
        },
        {
          "syntax": "git fork <url>",
          "description": "Oppretter en kopi av et eksternt repo på din egen GitHub-profil",
          "details": "<strong>git fork:</strong> Oppretter en kopi av et eksternt repo slik at du kan gjøre endringer uten å påvirke originalen."
        }
      ]
    }
  ]
}
```
## Bidrag
Bidrag til KodeList er alltid velkomne! Vi ønsker å lage en norsk huskeliste over kode. Hvis du har eksempler på liste-syntaks i andre programmeringsspråk eller andre nyttige kommandoer, er det bare å bidra ved å legge til flere eksempler og åpne en pull request.
