Python Math Kommando Huskeliste

Grunnleggende Operasjoner

math.pow()
Beregn potensverdier

Detaljert Forklaring

  • math.pow(x, y) beregner x opphøyd i y.

Eksempel

import math

result = math.pow(2, 3)
print("2 opphøyd i 3 er", result)

Denne funksjonen returnerer 8.0 som resultat.

math.gcd()
Finn største felles divisor (GCD)

Detaljert Forklaring

  • math.gcd(a, b) returnerer den største felles divisoren til a og b.

Eksempel

import math

gcd_value = math.gcd(48, 18)
print("GCD av 48 og 18 er", gcd_value)

Resultatet vil være 6.

math.log10()
Beregne logaritme med base 10

Detaljert Forklaring

  • math.log10(x) returnerer logaritmen til x med base 10.

Eksempel

import math

log_value = math.log10(1000)
print("Logaritmen til 1000 med base 10 er", log_value)

Returnerer 3.0.

Avanserte Beregninger

math.sqrt()
Beregne kvadratroten

Detaljert Forklaring

  • math.sqrt(x) returnerer kvadratroten av x.

Eksempel

import math

sqrt_value = math.sqrt(16)
print("Kvadratroten av 16 er", sqrt_value)

Resultatet blir 4.0.

math.factorial()
Beregne fakultet

Detaljert Forklaring

  • math.factorial(n) returnerer fakultetet til n.

Eksempel

import math

fact = math.factorial(5)
print("Fakultetet til 5 er", fact)

Returnerer 120.

Trigonometriske Funksjoner

math.sin()
Beregn sinus

Detaljert Forklaring

  • math.sin(x) returnerer sinusverdien til x (i radianer).

Eksempel

import math

sin_value = math.sin(math.pi / 2)
print("Sinus til pi/2 er", sin_value)

Resultatet blir 1.0.

math.cos()
Beregn cosinus

Detaljert Forklaring

  • math.cos(x) returnerer cosinusverdien til x (i radianer).

Eksempel

import math

cos_value = math.cos(0)
print("Cosinus til 0 er", cos_value)

Returnerer 1.0.

Konstantverdier

math.pi & math.e
Matematiske konstanter

Detaljert Forklaring

  • math.pi gir verdien av π.
  • math.e gir den matematiske konstanten e.

Eksempel

import math

print("Verdien av pi er", math.pi)
print("Verdien av e er", math.e)

Disse konstantene brukes ofte i matematiske beregninger.

Andre Funksjoner

math.floor() & math.ceil()
Rundingsfunksjoner

Detaljert Forklaring

  • math.floor(x) runder ned til nærmeste heltall.
  • math.ceil(x) runder opp til nærmeste heltall.

Eksempel

import math

print("Floor av 2.7 er", math.floor(2.7))
print("Ceil av 2.7 er", math.ceil(2.7))

Denne koden demonstrerer hvordan man runder tall.

math.trunc()
Fjerne desimaler

Detaljert Forklaring

  • math.trunc(x) fjerner desimaldelen av x uten avrunding.

Eksempel

import math

print("Trunc av 2.9 er", math.trunc(2.9))

Resultatet blir 2.
