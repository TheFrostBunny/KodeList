Grunnleggende Variabler og Datatyper

x = 5
Tildele et heltall til en variabel

Detaljert Forklaring

  • x får verdien 5 (en integer).

Eksempel

x = 5
print(x)

Denne syntaksen brukes til å definere og skrive ut heltallsvariabler i Python.

pi = 3.14
Tildele et desimaltall (float) til en variabel

Detaljert Forklaring

  • Variabelen pi settes til verdien 3.14, et flyttall.

Eksempel

pi = 3.14
print("Verdien av pi er:", pi)

Flyttall brukes når du trenger presisjon med desimalverdier.

melding = "Hei, verden!"
Definere en streng (str)

Detaljert Forklaring

  • Setter variabelen melding til en tekstverdi.

Eksempel

melding = "Hei, verden!"
print(melding)

Strenger brukes for tekstlig data i Python.

active = True
Definere en boolsk verdi

Detaljert Forklaring

  • Variabelen active settes til enten True eller False.

Eksempel

active = True
print("Er aktiv:", active)

Boolske verdier brukes for logiske tester og betinget utførelse.

Datastrukturer og Kontrollstrukturer

frukter = ["eple", "banan", "cherry"]
Opprette en liste

Detaljert Forklaring

  • En liste er en ordnet og endringsbar samling av elementer.

Eksempel

frukter = ["eple", "banan", "cherry"]
print(frukter)

Lister er nyttige for å lagre grupper av data under ett navn.

if ... elif ... else
Beslutningsstrukturer

Detaljert Forklaring

  • Lager en betinget blokk som utfører kode basert på ulike tilstander.

Eksempel

alder = 20
if alder >= 18:
    print("Voksen")
else:
    print("Ikke voksen")

Denne strukturen er essensiell for å lage dynamiske programmer.

for frukt in frukter:
For-løkke for iterasjon

Detaljert Forklaring

  • Itererer over hvert element i en liste.

Eksempel

for frukt in frukter:
    print(frukt)

For-løkker lar deg behandle alle elementer i en sekvens.

while teller < 3:
While-løkke

Detaljert Forklaring

  • Utfører en blokk med kode så lenge en betingelse er sann.

Eksempel

teller = 0
while teller < 3:
    print("Teller:", teller)
    teller += 1

While-løkker er nyttige når antall iterasjoner ikke er forhåndsbestemt.

Funksjoner og Lambda

def hils(navn):
Definere en funksjon

Detaljert Forklaring

  • Funksjoner defineres med nøkkelordet def og kan ta imot parametere.

Eksempel

def hils(navn):
    return "Hei " + navn

print(hils("Verden"))

Funksjoner bidrar til å organisere og gjenbruke kode.

lambda x: x * 2
Lambda-funksjon for enkle uttrykk

Detaljert Forklaring

  • Lambda-funksjoner er anonyme funksjoner for kortsiktige operasjoner.

Eksempel

dobbel = lambda x: x * 2
print(dobbel(5))

Denne syntaksen er nyttig når funksjonen er enkel og brukes en gang.

Klasser og Objekter

class Person:
Definere en klasse

Detaljert Forklaring

  • Klasser benyttes i objektorientert programmering for å modellere objekter.
  • Konstruktøren __init__ initialiserer klasseattributter.

Eksempel

class Person:
    def __init__(self, navn, alder):
        self.navn = navn
        self.alder = alder

    def hils(self):
        return "Hei, jeg heter " + self.navn

p = Person("Kari", 30)
print(p.hils())

Objektorientert programmering gir en strukturert tilnærming til koding.

Feilhåndtering og Moduler

try ... except
Håndtere unntak

Detaljert Forklaring

  • try brukes for å teste kode som kan kaste unntak, og except håndterer feilen.

Eksempel

try:
    resultat = 10 / 0
except ZeroDivisionError:
    print("Kan ikke dele med null!")

Bruk feilhåndtering for å gjøre programmene dine sikrere og mer robuste.

import math
Importere en modul

Detaljert Forklaring

  • Moduler gir deg tilgang til ekstra funksjonalitet utenfor standardbiblioteket.

Eksempel

import math
print("Kvadratroten av 9 er", math.sqrt(9))

Ved å importere moduler kan du utvide Python-programmenes funksjonalitet.

Dokumentasjon

""" Docstring """
Bruk av docstrings

Detaljert Forklaring

  • Docstrings benyttes for å dokumentere moduler, funksjoner og klasser i koden.
  • Følg PEP 257 for konsistente dokumentasjonsstandarder.

Eksempel

def legg_sammen(a, b):
    """
    Legger sammen to tall.
    
    Parametere:
        a (int/float): Første tall.
        b (int/float): Andre tall.
    
    Returnerer:
        int/float: Summen av a og b.
    """
    return a + b

God dokumentasjon gjør koden enklere å forstå og vedlikeholde.
